# Guitar Super System - Landing Page

Marketing landing page for Guitar Super System, a premier online guitar education platform.

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Design
- Dark theme matching existing brand
- Teal accent color (#1DB99A)
- Responsive design
- Inspired by Tailwind Plus UI blocks

## Sections
1. Hero with CTA
2. Features (3 cards)
3. Curriculum (4 course cards)
4. Pricing
5. Platform availability
6. Footer with social links

## Deployment
This site is designed for GitHub Pages.

```bash
# Push to GitHub and enable Pages in repo settings
git push origin main
```
